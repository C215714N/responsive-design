Introduccion a Bootstrap

Responsive Web Design

Bootstrap es un Framework originalmente creado por Twitter, que permite crear interfaces web con CSS y JavaScript, cuya particularidad es la de adaptar la interfaz del sitio web al tamaño del dispositivo en que se visualice. Es decir, el sitio web se adapta automáticamente al tamaño de una PC, una Tablet u otro dispositivo.

Este Framework posee definiciones de media queries y porcentajes en su CSS para hacer una web Responsive. Además, se basa en la simplicidad de sus interfaces, lo cual es una tendencia del mercado, en las que tiende a diseño plano, botones grandes, etc… para facilitar la usabilidad en los dispositivos más pequeño, atendiendo a la Experiencia de Usuario.

Ventajas

  • Se puede diseñar un sitio combinandolo con HTML, CSS y Javascript.
  • Incluye un sistema de grillas útil para maquetar por columnas.
  • Sus plantillas son de sencilla adaptación responsive.
  • Se integra facilmente con bibliotecas de JavaScript.
  • Cuenta con una documentación completa para su aprendizaje.
  • Facilita el diseño de interfaces y además incluye una plantilla optimizada.

Desventajas

  • Incluir el framework en un trabajo iniciado puede ser tedioso y molesto.
  • Es complicado cambiar de versión si realizaron modificaciones profundas.
  • No es ligero y para algunas funcionalidades se debe usar Javascript y jQuery.
  • Debe adaptarse el diseño a un sistema de grillas de 12 columnas.
  • Por defecto trae anchos, márgenes, altos de línea y realiza cambios específicos.

Responsive Breakpoints

Dispositivos y Media Queries

Los Breakpoints corresponden a medidas de anchura en las que se realizan cambios para el diseño responsive y aplican los estilos CSS concretos. Es un valor de ancho, generalmente en pixeles, que se toma en cuenta para fijar el límite entre dos dispositivos. Ayudan a planificar y adaptar un diseño base a cada dispositivo, pues tiene una dimensión a tomar en cuenta sobre la que trabajar.

Sistema de Grillas

Flexbox, Filas y Columnas

Una Grilla es una estructura conformada por líneas verticales y horizontales que se intersectan. En el diseño web sirven para colocar y alinear contenido de modo que el layout no luzca desordenado y hasta cierto punto para guiar la vista de los usuarios.

Bootstrap posee un modelo de 12 columnas invisibles para el usuario, que facilita el diseño y desarrollo de aplicaciones y sitios web. Son un punto de partida que permiten diagramar la página con el uso de módulos utilizando la clase col, opcionalmente el breakpoint, y el numero de columnas a ocupar.

Columnas xs Small Medium Large devices

El sistema de grillas se basa en contenedores flexibles que se escalan dependiendo del tamaño del dispositivo. Para ajustar los contenedores disponemos de diferentes clases:

  • .container el contenido aparece centrado y con un ancho fijo.
  • .container-fluid el contenido ocupa todo el ancho disponible (100%).

Desplazar columnas

En caso que necesitemos Compensar un espacio aumentando la separacion entre columnas o simplemente quisieramos desplazarla hacia la derecha, podemos utilizar la clase offset que basicamente aumenta el margen izquierdo en un valor igual al porcentaje del numero de columna seleccionado.
